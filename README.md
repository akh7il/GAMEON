<h2>📌 Overview</h2>

GameOn is a fully designed frontend gaming platform that brings multiple browser-based games into a single unified experience.

It is not just a collection of games — it is a structured gaming ecosystem with:
<ul>
  <li>Game hub interface</li>
  <li>Category-based filtering</li>
  <li>Individual game modules</li>
  <li>Authentication UI</li>
  <li>Contact system</li>
</ul>
